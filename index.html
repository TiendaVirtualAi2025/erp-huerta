Como conectar

1. Abrir editor

Google Sheets → Extensiones → Apps Script

2. Pegar el script

Borrar todo y pegar el archivo apps_script_v17.gs

3. Guardar e implementar

Ctrl+S → Implementar → Nueva implementacion → Aplicacion web
Ejecutar como: Yo · Acceso: Cualquier persona

4. Copiar URL

Pegar la URL /exec arriba → Probar Conexion → debe decir V17

5. Actualizar script

Implementar → Administrar → Editar → Nueva version → Implementar
