📋 Pasos para conectar

1 — Abrir editor

Google Sheets → Extensiones → Apps Script

2 — Pegar el código

Borra todo el editor y pega el archivo apps_script_v15.gs

3 — Implementar

Implementar → Nueva implementación → Aplicación web
Ejecutar como: Yo · Acceso: Cualquier persona

4 — Pegar URL aquí

Copia la URL, pégala arriba → toca Probar Conexión

5 — Actualizar script

Implementar → Administrar → Editar → Nueva versión

💡 Guía de uso

  • Al elegir cualquier cama en el formulario, huerta, cultivo y origen se rellenan solos desde tu Sheets
  • Las listas de fases, estados, acciones y destinos vienen de la columna Config — cámbialas en el Excel y se actualizan aquí
  • Al elegir un participante, se muestra su última asistencia registrada
  • Al elegir un insumo, se muestran stock actual, precio y unidad
  • El botón ↻ Sync recarga todo desde Sheets sin recargar la página
  • Los datos se guardan en caché local 30 min — funciona sin señal
